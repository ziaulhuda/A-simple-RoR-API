Comments API

API to handel Comment resource. Authorization required .

Deleting a comment

DELETE /posts/:post_id/comments/:id

Admin can delete any comment. "user" type users can only delete their own comments. Guests cannot delete any comment.

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: Pu1L6oEp1BkZwsdfcNbJDbVj
Host: example.org
Cookie:

Route

DELETE /posts/1/comments/4

cURL

curl "http://localhost:3000/posts/1/comments/4" -d '' -X DELETE \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: Pu1L6oEp1BkZwsdfcNbJDbVj" \
	-H "Host: example.org" \
	-H "Cookie: "

Response

Headers

Cache-Control: no-cache
X-Request-Id: 21988bb5-5117-4f5e-a9c0-374488df9e9d
X-Runtime: 0.010051

Status

204 No Content
