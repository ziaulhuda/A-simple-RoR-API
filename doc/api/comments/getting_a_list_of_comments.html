Comments API

API to handel Comment resource. Authorization required .

Getting a list of comments

GET /posts/:post_id/comments/

Response Fields

Name Description
id ID of the comment
body Contents of the comment
post_id ID of the post this comment is related to
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: ZepfwBq2ZwLRc3Yra8tUJ9yS
Host: example.org
Cookie:

Route

GET /posts/1/comments/

cURL

curl -g "http://localhost:3000/posts/1/comments/" -X GET \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: ZepfwBq2ZwLRc3Yra8tUJ9yS" \
	-H "Host: example.org" \
	-H "Cookie: "
