Comments API

API to handel Comment resource. Authorization required .

Creating a comment

POST /posts/:post_id/comments/

Admin and "user" can create comments. Guest users cannot create a comment.

Parameters

Name Description
body Contents of the comments

Response Fields

Name Description
id ID of the comment
body Contents of the comment
post_id ID of the post this comment is related to
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: 5UM5X5dMvRrdrKJiWjj32xWT
Host: example.org
