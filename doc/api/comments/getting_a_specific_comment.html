Comments API

API to handel Comment resource. Authorization required .

Getting a specific comment

GET /posts/:post_id/comments/:id

Response Fields

Name Description
id ID of the comment
body Contents of the comment
post_id ID of the post this comment is related to
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: 85JNnnuphyuJ4vS746u8yEq3
Host: example.org
Cookie:

Route

GET /posts/1/comments/4

cURL

curl -g "http://localhost:3000/posts/1/comments/4" -X GET \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: 85JNnnuphyuJ4vS746u8yEq3" \
	-H "Host: example.org" \
