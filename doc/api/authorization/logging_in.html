Authorization API

Authorization service

Logging in

POST /signin

Parameters

Name Description
email Email of the user
password Password of the user

Response Fields

Name Description
id ID of the user
auth_token Authorization token to be sent in request headers for using other API actions
name Name of the user
role Role of the user. One of [admin, user, guest]

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

Route

POST /signin

Body

{"email":"lemuel@krajcikmarks.name","password":"12345678"}

cURL

curl "http://localhost:3000/signin" -d '{"email":"lemuel@krajcikmarks.name","password":"12345678"}' -X POST \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Host: example.org" \
	-H "Cookie: "
