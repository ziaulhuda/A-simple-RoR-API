Posts API

API to handel Post resource. Authorization required .

Updating a post

PUT /posts/:id

Admin can update any post. "user" type users can only update their own posts. Guests cannot update any post.

Parameters

Name Description
title Title of the post
body Contents of the post

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: L8svFJHjXkwj2wxd2NcTVJN6
Host: example.org
Cookie:

Route

PUT /posts/4

Body

{"title":"A wonderful post"}

cURL

curl "http://localhost:3000/posts/4" -d '{"title":"A wonderful post"}' -X PUT \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: L8svFJHjXkwj2wxd2NcTVJN6" \
	-H "Host: example.org" \
	-H "Cookie: "

Response

Headers

Cache-Control: no-cache
X-Request-Id: d5cbebf2-f4f9-4359-90cb-b44967516cc0
X-Runtime: 0.007266

Status

204 No Content
