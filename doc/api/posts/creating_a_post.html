Posts API

API to handel Post resource. Authorization required .

Creating a post

POST /posts

Admin and "user" can create posts. Guest users cannot create a post.

Parameters

Name Description
title Title of the post
body Contents of the post

Response Fields

Name Description
id ID of the post
title Title of the post
body Contents of the post
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: pdudv2QNZsyiJ4KP774pkUiD
Host: example.org
