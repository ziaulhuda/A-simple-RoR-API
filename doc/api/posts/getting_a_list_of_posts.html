Posts API

API to handel Post resource. Authorization required .

Getting a list of posts

GET /posts

Response Fields

Name Description
id ID of the post
title Title of the post
body Contents of the post
user_id ID of the creator

Headers

Accept: application/json
Content-Type: application/json
Authorization: ixx7TF5f8ktHZsrkFj4arExt
Host: example.org
Cookie:

Route

GET /posts
