Posts API

API to handel Post resource. Authorization required .

Getting a list of posts

GET /posts

Response Fields

Name Description
id ID of the post
title Title of the post
body Contents of the post
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: ECnwCE57fTCaeE9UQxKWDrb5
Host: example.org
Cookie:

Route

GET /posts

cURL

curl -g "http://localhost:3000/posts" -X GET \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: ECnwCE57fTCaeE9UQxKWDrb5" \
	-H "Host: example.org" \
	-H "Cookie: "
