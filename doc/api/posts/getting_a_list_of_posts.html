Posts API

API to handel Post resource. Authorization required .

Getting a list of posts

GET /posts

Response Fields

Name Description
id ID of the post
title Title of the post
body Contents of the post
user_id ID of the creator

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: KgWVkLyLotBBmn5yh8FdrXgi
Host: example.org
Cookie:

Route

GET /posts

curl -g "http://localhost:3000/posts" -X GET \

	-H "Content-Type: application/json" \
