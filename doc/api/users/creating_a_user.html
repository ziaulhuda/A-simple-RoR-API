Users API

API to handel User resource. Authorization required.

Creating a user

POST /users

Admin user can create all types of users. "user" type users can only create "guest" type users. Guest users cannot create a user.

Parameters

Name Description
name Name of the user
email Email of the user
password Password of the user
role user role ["admin", "user", "guest"(default)]. Only admin can provide
disabled Is the user disabled from logging in? Only admin can provide

Response Fields

Name Description
id ID of the user
name Name of order
email Email of the user
role Role of the user (Only if the logged in user is admin)
disabled Has user been disabled (Only if the logged in user is admin)
