Users API

API to handel User resource. Authorization required.

Getting a list of users

GET /users

Response Fields

Name Description
id ID of the user
name Name of the user
email Email of the user
role Role of the user (Only if the logged in user is admin)
disabled Has user been disabled (Only if the logged in user is admin)

Request

Headers

Accept: application/json
Content-Type: application/json
Authorization: TSJpN5h32JiPHu1KNUHNAgwx
Host: example.org
Cookie:

Route

GET /users

cURL

curl -g "http://localhost:3000/users" -X GET \
	-H "Accept: application/json" \
	-H "Content-Type: application/json" \
	-H "Authorization: TSJpN5h32JiPHu1KNUHNAgwx" \
	-H "Host: example.org" \
	-H "Cookie: "
